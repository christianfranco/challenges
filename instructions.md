Setting up
==========
`$ bundle install`

Running
=======
Run with  
`$ shotgun`

Testing
=======
`$ rake jasmine`  
Open the provided address on the browser.

Or via console:  
`$ rake jasmine:ci`

**Note:** JST's are hardcoded. Update your `jst.js` file when changing templates. Got a fix for this? Extra points!;-)
